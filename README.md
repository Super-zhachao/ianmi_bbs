##使用说明:
**step0：**
安装安米程序，官方程序下载地址：[安米程序](http://bbs.ianmi.com)
**step1：**
把文件解压内容到根目录覆盖
**step2：**
修改数据表setting的theme字段为ianmi_bbs
**step3：**
刷新查看效果

**tip:
因为是基于官方模板修改，请勿移除官方自带的静态资源文件！！删除后可能会造成模板乱版！！**
